Simplex Pattern Library

Simplex is a simple, scalable framework that can be used as a base for complex websites and web applications. The word "simplex" comes from the Latin for "simple".

Simplex uses BEM and OOCSS principles, and aims is to:

  • define base HTML markup
  • define element styles
  • define flexible form and table markup and styling
  • define a series of re-usable modules, modifiers and sub-modules
  • define a series of helper classes
  • define a simple grid system that can be adjusted as needed

Helpers

Helper classes can be used to tweak aspects of the layout without having to write specific CSS components.

Modules

Modules are re-usable components that have been abstracted down to a single purpose. They are completely "de-coupled" from the HTML - which means that they can be placed anywhere within web pages - regardless of any parent elements or classes.
